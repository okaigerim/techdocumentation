Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

What You Should Already Know

This guide assumes that you have a basic understanding of programming concepts and are familiar with fundamental programming terminologies.

Python vs. Other Languages

Python is often compared with other programming languages, but it has some unique features:

  • Python emphasizes code readability with its clean and easily understandable syntax.
  • Unlike languages like Java, Python doesn't require explicit type declarations for variables.
  • Python's dynamic typing allows variables to change types during runtime.
Hello World

To start with Python, you can write a simple "Hello, World!" program:

print("Hello, World!")

This program uses the print function to display the text "Hello, World!" in the console.

Variables

In Python, variables are used to store data. You don't need to specify the data type explicitly; Python infers it based on the assigned value.

x = 42 name = "Alice"
Data Types

Python supports various data types, including:

  • Integers: Whole numbers (e.g., 5, -2).
  • Floats: Decimal numbers (e.g., 3.14, -0.5).
  • Strings: Text (e.g., "Hello, Python!").
  • Booleans: True or False values.
  • Lists: Ordered collections of items.
  • Dictionaries: Key-value pairs for storing data.
  • Sets: Unordered collections of unique items.
  • and more...
Conditional Statements: if...else

Use the if...else statement to perform different actions based on conditions.

age = 18 if age >= 18: print("You are an adult.") else: print("You are a minor.")
Loops: while Statement

The while loop executes a block of code repeatedly while a condition is true.

count = 0 while count < 5: print("Count:", count) count += 1
Functions

Functions are blocks of code that can be reused. They help in organizing and structuring your code.

def greet(name): return "Hello, " + name + "!" print(greet("Alice")) # Output: Hello, Alice!
